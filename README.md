<h1>Clove Dental – Frontend</h1>

<p>
This is a responsive dentist landing page where patients can book appointments online.
The design is clean, simple, and optimized for all screen sizes.
</p>

<div>
  <code>#HTML</code>
  <code>#CSS</code>
  <code>#JavaScript</code>
  <code>#API</code>
</div>

<hr>

<h2>Features</h2>
<ul>
  <li>Fully responsive design (Mobile, Tablet, Desktop)</li>
  <li>Appointment booking form</li>
  <li>Client-side form validation</li>
  <li>Fetch API integration with backend</li>
  <li>Success page with booking ID and appointment date</li>
</ul>

<h2>Technologies Used</h2>
<ul>
  <li>HTML5</li>
  <li>CSS3</li>
  <li>Vanilla JavaScript</li>
  <li>Fetch API</li>
</ul>

<h2>How It Works</h2>
<ul>
  <li>User fills appointment form</li>
  <li>JavaScript validates inputs</li>
  <li>Form data is sent to backend API</li>
  <li>On success, user is redirected to success page</li>
</ul>

<h2>API Endpoint Used</h2>
<p><code>POST /api/send-email</code></p>
